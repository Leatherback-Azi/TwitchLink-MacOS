마루 엔진


영상의 손실을 최소화하고 안정적인 데이터를 받기 위한 특수한 사용에 적합합니다.


비디오 목록의 영상만 다운로드할 수 있는 엔진입니다.
음소거 부분의 데이터를 최대한 복구하고 영상의 데이터 손실을 최소로 줄입니다.
초고속 다운로드 기능을 지원합니다.
부분 다운로드를 지원하지 않습니다.
다운로드와 인코딩 작업이 순차적으로 진행됩니다.
다운로드와 인코딩을 별개로 처리하여 최종 영상의 용량만큼 임시 저장 공간이 추가로 필요합니다.
다운로드 도중 취소 시 다운로드된 임시 데이터를 삭제합니다.
인코딩 도중 취소 시 해당 시점까지 인코딩된 영상이 출력됩니다.

 

초고속 다운로드

데이터를 초고속으로 다운로드합니다.
트래픽 소모가 심하며 유튜브나 트위치 등 네트워크를 사용하는 다른 작업에 버퍼링을 유발할 수 있습니다.

 

업데이트 추적 모드

변동되는 영상의 최종 데이터까지 다운로드합니다.
실시간으로 저장되는 생방송 다시보기는 추가되는 영상 데이터를 이어서 받아옵니다.
다시보기가 저장되는 생방송은 다시보기 목록에서 해당 생방송을 선택함으로써 다운로드 가능합니다.
다운로드 완료 후 추가 영상파일을 확인하는 작업이 5분간 이어집니다.
생방송 도중 발생하는 버퍼링, 끊김 등의 영향을 받지 않고 서버에 안정적으로 저장된 데이터를 다운로드합니다.

 

일반 다운로드 모드

다운로드 시점의 영상을 그대로 다운로드합니다.
실시간으로 저장되는 생방송 다시보기는 해당 시점까지의 영상만 다운로드합니다.
다운로드 완료 후 바로 인코딩 작업이 진행됩니다.
생방송 도중 발생하는 버퍼링, 끊김 등의 영향을 받지 않고 서버에 안정적으로 저장된 데이터를 다운로드합니다.
